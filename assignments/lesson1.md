# Lesson 1 
## Intro
### Task 1

###### 1 point

A = {1, 2}
B = {1, 4, 8, 12}

Найдите элементы множеств:
1. A ∪ (B \ A)
2. A \ (B ∪ A)
3. A ⋂ (B \ A)

### Task 2

###### 2 points

Докажите (A ⊆ B) ⇒ (A \ B = ∅)

### Task 3

###### 3 points

Докажите (A \ B = ∅) ⇒ (A ⊆ B)

### Task 4

###### 2 points

Какие пары следующих форм являются равносильными?
1. F1(x) = x + 1
2. F2(x, y) = x + y - y
3. F3(x, y) = x + y/y 
4. F4(x, y) = x
